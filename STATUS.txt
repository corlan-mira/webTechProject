 PROJECT VISION
────────────────────────────────────────────────────────────────

Event Attendance Monitoring System:
  A professional web application for managing event attendance
  with dual check-in mechanisms (text codes + QR codes),
  real-time tracking, and data export capabilities.

Target Users:
  • Event Organizers - Manage events and attendance
  • Participants - Quick and easy check-in

Technology:
  • React SPA for frontend
  • Node.js/Express REST API
  • PostgreSQL relational database
  • Sequelize ORM

Deployment:
  • Multiple deployment options
  • Production-ready architecture
  • Security-first approach
  • Scalable design
